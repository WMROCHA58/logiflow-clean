Política de Privacidade

O LogiFlow coleta apenas os dados necessários para o funcionamento do aplicativo.

Dados coletados

  • Localização do motorista
  • Endereços de entrega
  • Telefone do cliente
  • Fotos de prova de entrega
  • Assinatura do recebedor

Uso dos dados

Os dados são utilizados exclusivamente para o gerenciamento de entregas dentro do aplicativo LogiFlow.

Essas informações não são vendidas, compartilhadas ou distribuídas para terceiros.
